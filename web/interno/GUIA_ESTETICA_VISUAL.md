# GUÍA ESTÉTICA VISUAL  
## Diagramas e imágenes conceptuales — Web Wild  
Quercus Flumen

---

# 1. Formato base

Todas las piezas visuales conceptuales seguirán este formato:

• Horizontal 16:9  
• Fondo blanco puro (#FFFFFF)  
• Resolución alta (mínimo 2000px ancho para web retina)  
• Márgenes amplios  
• Espacios respirables  

Wild comunica calma y seguridad.  
La imagen nunca debe sentirse saturada.

---

# 2. Estilo general

Estética mixta:

• 50% técnico  
• 50% emocional  

Debe combinar:

✔ Diagrama claro  
✔ Elementos naturales sutiles  
✔ Jerarquía tipográfica limpia  
✔ Minimalismo elegante  

No usar:

✘ Gradientes agresivos  
✘ Texturas recargadas  
✘ Colores saturados  
✘ Iconografía infantil  
✘ Efectos 3D exagerados  

---

# 3. Paleta cromática oficial

Colores base:

Negro profundo  
Ámbar Wild  
Verde bosque suave  
Gris técnico muy claro  

El blanco debe dominar siempre.

El ámbar se utiliza como punto focal, no como fondo masivo.

---

# 4. Tipografía (criterios)

• Títulos: elegantes, limpios, contundentes  
• Subtítulos: ligeros, técnicos  
• Frases finales: breves, directas  

Regla:

Menos texto, más impacto.

No más de 3 bloques de texto por imagen.

---

# 5. Estructura estándar de cada pieza

Cada imagen horizontal debe contener:

1. Título emocional fuerte  
2. Subtítulo técnico breve  
3. Diagrama central claro  
4. Frase final corta de cierre  

Ejemplo estructural:

TÍTULO  
Subtítulo técnico  

[ Diagrama limpio ]

Frase final.

---

# 6. Jerarquía visual

El orden de lectura debe ser:

1. Título  
2. Diagrama  
3. Frase final  

Nunca debe haber confusión visual.

El ojo debe recorrer la pieza de izquierda a derecha sin fricción.

---

# 7. Coherencia de marca

Todas las piezas deben transmitir:

• Seguridad  
• Equilibrio  
• Madurez  
• Profesionalidad  
• Naturaleza controlada  

Nunca transmitir:

• Urgencia  
• Exageración  
• Competencia agresiva  
• Comparativa directa  

---

# 8. Filosofía visual

Wild no necesita gritar.

Wild respira.

El blanco es parte del mensaje.  
El espacio es parte del equilibrio.

Cada imagen debe sentirse como una página de un libro técnico elegante, no como un anuncio.
