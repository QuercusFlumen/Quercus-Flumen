# PASTEURIZATION_CCP.md
Quercus Flumen – Wild V1
Control Crítico de Proceso (CCP)

Versión: 1.0  
Estado: Implementación obligatoria en lotes comerciales  
Aplicable a: Lotes ≥ 200 L

------------------------------------------------------------
1. OBJETIVO
------------------------------------------------------------

Definir el Control Crítico de Proceso (CCP) correspondiente a la fase de pasteurización térmica del extracto Wild V1, garantizando:

- Reducción significativa de carga microbiana.
- Estabilidad microbiológica en almacenamiento.
- Ausencia de fermentación secundaria.
- Mantenimiento de integridad orgánica (taninos / fracción fúlvica).

------------------------------------------------------------
2. PRINCIPIO TÉCNICO
------------------------------------------------------------

La pasteurización aplicada no busca esterilización absoluta, sino:

• Inactivación de bacterias vegetativas.
• Reducción de levaduras y hongos.
• Interrupción de actividad enzimática residual.

Se evita sobrecalentamiento para no degradar fracción fúlvica.

------------------------------------------------------------
3. PARÁMETROS OPERATIVOS ESTÁNDAR
------------------------------------------------------------

Método: Pasteurización térmica en tanque cerrado con agitación suave.

Temperatura objetivo: 65–72 °C  
Tiempo mínimo efectivo: 30 minutos una vez alcanzada T objetivo.

Rango validado recomendado:
- 68 °C ± 2 °C durante 30 min

No superar:
- 75 °C (riesgo degradación orgánica)
- 40 minutos sostenidos

------------------------------------------------------------
4. CONTROL Y MEDICIÓN
------------------------------------------------------------

Instrumentación mínima requerida:

- Termómetro digital calibrado (±0.5 °C)
- Sonda sumergida en zona media del volumen
- Registro manual o digital de temperatura cada 5 minutos

Registro obligatorio:

| Tiempo | Temperatura (°C) |
|--------|------------------|
| T0     |                  |
| +5m    |                  |
| +10m   |                  |
| ...    |                  |

Criterio de aceptación:

✔ Temperatura ≥ 65 °C durante al menos 30 min continuos  
✔ No superar 75 °C  
✔ Enfriamiento controlado < 40 °C en menos de 90 min

------------------------------------------------------------
5. PUNTO CRÍTICO DE CONTROL (CCP)
------------------------------------------------------------

Falla crítica si:

- No se alcanza 65 °C
- Tiempo efectivo < 30 min
- Se supera 75 °C

Acción correctiva:

- Repetir ciclo térmico completo.
- Si se detecta olor anómalo → lote en observación 72h.
- Si hay precipitación anormal → evaluación físico-química.

------------------------------------------------------------
6. VALIDACIÓN POST-PASTEURIZACIÓN
------------------------------------------------------------

Tras enfriamiento:

Medir:
- pH
- Conductividad (µS/cm)
- TDS (ppm)
- Observación visual
- Olor

Criterio esperado (según lote base):

- Variación pH ≤ ±0.15
- Variación µS ≤ ±5%
- Sin aparición de sedimentos
- Olor neutro / bosque sutil

------------------------------------------------------------
7. RIESGOS CONTROLADOS
------------------------------------------------------------

La pasteurización controlada evita:

- Fermentación espontánea
- Crecimiento fúngico
- Producción de gases
- Oscilaciones de pH por metabolismo bacteriano

------------------------------------------------------------
8. OBSERVACIÓN FILOSÓFICA DEL PROCESO
------------------------------------------------------------

Este paso no industrializa el producto.
Estabiliza la naturaleza sin alterar su estructura química esencial.
