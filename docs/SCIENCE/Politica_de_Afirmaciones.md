# Politica_de_Afirmaciones.md
Quercus Flumen – Marco Formal de Afirmaciones Técnicas

Versión: 1.0  
Estado: Activo  
Aplicable a: Todos los documentos técnicos, comerciales y científicos del proyecto.

------------------------------------------------------------
1. OBJETIVO
------------------------------------------------------------

Establecer un marco claro, transparente y verificable que distinga entre:

• Datos medidos directamente  
• Parámetros inferidos mediante proxies  
• Hipótesis técnicas en validación futura  

Este documento garantiza coherencia científica y evita sobreafirmaciones no verificadas.

------------------------------------------------------------
2. PRINCIPIOS RECTORES
------------------------------------------------------------

Quercus Flumen se rige por tres principios:

1) No afirmar como medido lo que es inferido.  
2) No afirmar como confirmado lo que es hipótesis.  
3) Diferenciar claramente marketing de ciencia.

La credibilidad técnica es prioritaria frente al impacto comercial.

------------------------------------------------------------
3. CLASIFICACIÓN DE AFIRMACIONES
------------------------------------------------------------

Todas las afirmaciones técnicas del proyecto se clasifican en tres niveles:

------------------------------------------------------------
NIVEL 1 – MEDIDO (✔)
------------------------------------------------------------

Datos obtenidos mediante instrumentación directa.

Incluye:

- pH (potenciómetro calibrado)
- Conductividad (µS/cm)
- TDS (ppm)
- Temperatura
- Observación visual documentada
- Olor organoléptico

Estos valores deben:

• Incluir instrumento utilizado  
• Indicar calibración  
• Preferiblemente incluir repetición (n≥3)  

Ejemplo correcto:

✔ “Conductividad media 235 µS/cm (n=3, ±3 µS)”

------------------------------------------------------------
NIVEL 2 – INFERIDO (🔶)
------------------------------------------------------------

Parámetros no medidos directamente pero deducidos mediante relación técnica coherente.

Ejemplos:

🔶 Baja liberación mineral secundaria (inferido por estabilidad µS)  
🔶 Baja fracción fermentable (inferido por estabilidad pH + olor)  
🔶 Alta fracción orgánica soluble (inferido por color + estabilidad)  

Estas afirmaciones deben incluir la expresión:

“Se infiere que…”  
“Indicios compatibles con…”  
“Hipótesis basada en estabilidad observada…”

Nunca deben presentarse como confirmación analítica.

------------------------------------------------------------
NIVEL 3 – HIPÓTESIS (🧠)
------------------------------------------------------------

Modelos teóricos pendientes de validación instrumental.

Ejemplos:

🧠 DOC estimado 180–280 mg/L  
🧠 Taninos + ácidos húmicos/fúlvicos 400–600 mg/L  
🧠 Perfil de microelementos complejados  

Estas afirmaciones:

• No constituyen especificación comercial.  
• No deben figurar en etiquetado.  
• Solo aplican a modelado técnico interno.  

Deben estar explícitamente marcadas como:

“Modelo teórico pendiente de validación ICP/TOC.”

------------------------------------------------------------
4. PROHIBICIONES EXPLÍCITAS
------------------------------------------------------------

Queda prohibido:

✘ Afirmar concentración de compuestos específicos sin análisis instrumental.  
✘ Usar el término “certificado” sin certificación externa.  
✘ Afirmar beneficios terapéuticos en peces.  
✘ Presentar hipótesis como especificaciones garantizadas.

------------------------------------------------------------
5. DECLARACIÓN SOBRE BENEFICIOS BIOLÓGICOS
------------------------------------------------------------

Los efectos atribuidos a taninos y ácidos húmicos se basan en literatura científica general.

En Quercus Flumen:

✔ Se mide estabilidad físico-química.  
🔶 Se infiere capacidad moduladora orgánica.  
🧠 Se modela perfil húmico-fúlvico.

Hasta disponer de análisis externo (ICP-OES, TOC, UV-Vis):

No se publicarán cifras absolutas como especificación contractual.

------------------------------------------------------------
6. COMPROMISO DE TRANSPARENCIA
------------------------------------------------------------

El proyecto adopta una postura de:

• Rigor técnico incremental.  
• Validación progresiva.  
• Transparencia documental abierta.  

Cada avance instrumental será documentado en:

docs/SCIENCE/Analisis_Instrumentales/

------------------------------------------------------------
7. CONCLUSIÓN
------------------------------------------------------------

La fortaleza de Quercus Flumen no reside en exagerar resultados, sino en:

• Medir lo que puede medirse.  
• Inferir con prudencia.  
• Modelar con honestidad.  

La credibilidad científica es un activo estratégico del proyecto.

Fin del documento.
