# Plan_Analitico_Instrumental.md
Quercus Flumen – Hoja de Ruta Analítica Instrumental

Versión: 1.0  
Estado: Plan estratégico de validación  
Aplicable a: Lotes ≥ 1000 L

------------------------------------------------------------
1. OBJETIVO
------------------------------------------------------------

Definir el marco analítico instrumental necesario para:

• Validar modelos teóricos actuales.
• Confirmar perfil orgánico real del extracto.
• Diferenciar científicamente Wild frente a extractos minerales.
• Establecer especificaciones técnicas futuras.

Este plan marca el paso de inferencia empírica a validación cuantitativa.

------------------------------------------------------------
2. NIVELES DE VALIDACIÓN
------------------------------------------------------------

Fase 1 – Validación Orgánica Básica  
Fase 2 – Perfil Mineral Completo  
Fase 3 – Caracterización Espectral y Fraccionamiento

------------------------------------------------------------
3. FASE 1 – VALIDACIÓN ORGÁNICA BÁSICA
------------------------------------------------------------

3.1 TOC / DOC (Carbono Orgánico Total / Disuelto)

Método: Analizador TOC por combustión catalítica o UV-persulfato.

Objetivo:

✔ Confirmar rango real de carbono orgánico.
✔ Estimar carga orgánica disuelta.
✔ Correlacionar con color y estabilidad.

Modelo actual (hipótesis):
DOC estimado 180–280 mg/L

Validación:

• Si DOC >150 mg/L → Modelo coherente.
• Si DOC <100 mg/L → Recalibrar estimación orgánica.

------------------------------------------------------------
4. FASE 2 – PERFIL MINERAL (ICP-OES)
------------------------------------------------------------

4.1 ICP-OES (Espectrometría de emisión óptica con plasma acoplado)

Analitos recomendados:

K, Ca, Mg, Fe, Mn, Zn, Cu, Na, Al, Si, P

Objetivos:

✔ Determinar carga mineral real.
✔ Confirmar baja mineralización estructural.
✔ Comparar con extractos de leonardita.

Modelo actual (hipótesis):

K: 5–40 mg/L  
Ca: 1–15 mg/L  
Mg: 1–10 mg/L  
Fe: 0.05–1 mg/L  

Interpretación clave:

Si carga mineral total es baja en relación con DOC,
se confirma predominancia orgánica sobre mineral.

------------------------------------------------------------
5. FASE 3 – CARACTERIZACIÓN ESPECTRAL
------------------------------------------------------------

5.1 UV-Vis (200–800 nm)

Objetivos:

✔ Detectar picos característicos de compuestos húmicos.
✔ Calcular índice E4/E6 (indicador fracción húmica vs fúlvica).
✔ Determinar grado de humificación.

Interpretación:

E4/E6 alto → mayor fracción fúlvica (más soluble).  
E4/E6 bajo → mayor fracción húmica pesada.

Este análisis es crítico para diferenciar Wild de leonardita industrial.

------------------------------------------------------------
6. ANÁLISIS COMPLEMENTARIOS
------------------------------------------------------------

6.1 pH buffer capacity test
Determinar capacidad tampón real frente a adición de ácido/base.

6.2 DBO (Demanda Biológica de Oxígeno)
Evaluar fracción biodegradable potencial.

6.3 Turbidez instrumental (NTU)
Correlacionar con estabilidad visual.

------------------------------------------------------------
7. MARCO COMPARATIVO CON LEONARDITA INDUSTRIAL
------------------------------------------------------------

Analizar en paralelo:

• Extracto leonardita comercial estándar.
• Wild V1.

Comparar:

| Parámetro | Leonardita | Wild |
|----------|------------|------|
| DOC      |            |      |
| ICP carga mineral |    |      |
| E4/E6    |            |      |
| Turbidez |            |      |
| Precipitación |      |      |

Objetivo:

Determinar si Wild presenta:

✔ Mayor proporción de fracción fúlvica soluble.
✔ Menor carga mineral.
✔ Menor precipitación.
✔ Mayor estabilidad estructural.

------------------------------------------------------------
8. ESTRATEGIA DE IMPLEMENTACIÓN
------------------------------------------------------------

Año 2026:
- TOC + ICP básico en lote 1000 L.

Año 2027:
- UV-Vis + índice E4/E6.

Año 2028:
- Estudio comparativo formal publicable.

------------------------------------------------------------
9. CRITERIO DE ÉXITO
------------------------------------------------------------

El modelo Wild quedará validado si:

✔ DOC confirma alta carga orgánica.
✔ ICP confirma baja mineralización relativa.
✔ UV-Vis indica fracción soluble predominante.
✔ Sedimentación <1 mm a 72h.

------------------------------------------------------------
10. FILOSOFÍA DEL PLAN
------------------------------------------------------------

Este plan no busca “probar que es perfecto”.

Busca:

• Medir.
• Comparar.
• Ajustar.
• Consolidar.

La fortaleza del proyecto no está en afirmar,
sino en validar progresivamente.
