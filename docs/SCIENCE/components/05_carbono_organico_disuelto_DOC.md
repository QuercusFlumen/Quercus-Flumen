# Carbono orgánico disuelto (DOC) en sistemas acuáticos

## Introducción

El carbono orgánico disuelto (DOC, Dissolved Organic Carbon) es uno de
los parámetros más utilizados en ciencia del agua para cuantificar la
cantidad de materia orgánica presente en solución.

A diferencia de otros conceptos más complejos como la materia orgánica
disuelta (DOM), el DOC representa una **medida cuantificable** del carbono
contenido en los compuestos orgánicos disueltos.

Esto lo convierte en una herramienta clave para conectar la química del
agua con datos analíticos reales.

---

## Qué es el DOC

El DOC se define como la cantidad de carbono presente en compuestos
orgánicos disueltos en el agua.

Se expresa normalmente en:

- mg/L de carbono orgánico

Incluye carbono procedente de:

- compuestos fenólicos
- taninos
- ácidos húmicos y fúlvicos
- otros compuestos orgánicos disueltos

No distingue entre tipos de moléculas, sino que mide el total de carbono
orgánico presente.

---

## DOC vs DOM

Es importante diferenciar:

DOM (Dissolved Organic Matter):

- conjunto de compuestos orgánicos disueltos
- concepto químico complejo
- incluye múltiples familias de moléculas

DOC (Dissolved Organic Carbon):

- fracción cuantificable de la DOM
- medida analítica
- representa solo el carbono

Esto implica que:

DOC = cantidad  
DOM = composición

---

## Origen del DOC en sistemas forestales

En entornos forestales, el DOC proviene principalmente de:

- lixiviación de hojas
- degradación de materia vegetal
- actividad microbiana del suelo

El flujo general puede describirse como:

bosque → hojarasca → transformación → DOM → DOC

Este proceso introduce carbono orgánico en el sistema acuático.

---

## Qué nos dice el DOC

El DOC permite estimar:

- carga orgánica del sistema
- disponibilidad de carbono para microorganismos
- nivel de complejidad orgánica del agua

En términos generales:

mayor DOC → mayor presencia de materia orgánica

---

## Limitaciones del DOC

Aunque es un parámetro muy útil, el DOC tiene limitaciones importantes.

No informa sobre:

- tipo de compuestos presentes
- reactividad química
- estructura molecular
- origen exacto del carbono

Dos aguas con el mismo DOC pueden ser químicamente muy diferentes.

---

## Relación con la microbiología

El DOC es una fuente de carbono para microorganismos heterótrofos.

Puede influir en:

- crecimiento bacteriano
- actividad metabólica
- dinámica del bucle microbiano

Sin embargo, no todo el DOC es igualmente utilizable:

- compuestos simples → mayor biodisponibilidad
- compuestos complejos → menor accesibilidad

---

## Relación con la química del agua

El DOC puede participar en:

- reacciones redox
- interacción con metales
- formación de complejos orgánicos

También puede influir indirectamente en:

- pH
- conductividad (de forma limitada)
- estabilidad del sistema

---

## DOC en sistemas de baja mineralización

En aguas muy blandas, donde la carga mineral es baja, el DOC adquiere
mayor relevancia.

En estos sistemas:

- la fracción orgánica tiene más peso relativo
- la química del agua depende más de la DOM
- pequeñas variaciones pueden tener mayor impacto

---

## DOC y color del agua

Existe una relación entre DOC y color, pero no es directa.

- algunos compuestos con alto DOC apenas colorean
- otros compuestos colorean intensamente con menor contribución al DOC

Esto implica que:

color ≠ DOC

---

## DOC como herramienta analítica

El DOC se mide en laboratorio mediante:

- oxidación catalítica a CO₂
- detección infrarroja
- analizadores TOC/DOC

Es un parámetro estándar en:

- limnología
- hidrología
- ciencia ambiental

---

## Relevancia para sistemas controlados

En acuarios y sistemas artificiales, el DOC puede utilizarse para:

- comparar diferentes fuentes de materia orgánica
- evaluar aportes orgánicos
- monitorizar cambios en el sistema

Sin embargo, siempre debe interpretarse junto con otros parámetros.

---

## Idea clave

El DOC no describe qué hay en el agua, sino cuánto carbono orgánico hay.

Es una herramienta cuantitativa, no cualitativa.

---

## Conclusión

El carbono orgánico disuelto (DOC) es un parámetro fundamental para
cuantificar la materia orgánica en sistemas acuáticos.

Permite conectar conceptos complejos como la DOM con datos medibles en
laboratorio, facilitando la comparación entre sistemas y el análisis de
procesos.

No obstante, su interpretación requiere contexto, ya que no proporciona
información sobre la composición ni la funcionalidad de los compuestos
presentes.

El DOC debe entenderse como una herramienta clave dentro de un conjunto
más amplio de análisis para comprender la química del agua.
