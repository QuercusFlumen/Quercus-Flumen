# Bibliografía base de SCIENCE/components

## Introducción

Este documento reúne la bibliografía base utilizada como marco científico
de referencia para la carpeta `SCIENCE/components`.

Su objetivo es proporcionar una base sólida para el estudio de:

- componentes orgánicos de origen forestal
- materia orgánica disuelta (DOM)
- variabilidad química de la hoja
- transformación de compuestos en agua
- implicaciones en sistemas acuáticos
- métodos de caracterización analítica

La bibliografía se organiza por bloques temáticos para facilitar su uso
como referencia científica y como apoyo a futuros desarrollos del proyecto.

## 8. Observación final

Esta bibliografía no pretende cerrar el tema, sino establecer un núcleo
de referencias de alta relevancia que permita:

- sostener científicamente la carpeta `SCIENCE/components`
- dar coherencia bibliográfica al trabajo ya realizado
- servir como base para futuras ampliaciones y validaciones experimentales

La lógica de esta selección es integrar tres niveles:

- ecología forestal
- química de la materia orgánica
- comportamiento de la DOM en agua
