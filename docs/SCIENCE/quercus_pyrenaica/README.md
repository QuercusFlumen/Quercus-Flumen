# Quercus pyrenaica — sistema forestal, hoja y transferencia al agua

## Visión

Esta sección estudia *Quercus pyrenaica* como un sistema completo.

No se analiza únicamente la hoja, sino el conjunto de procesos que la generan, transforman y conectan con el medio acuático.

La hoja es una consecuencia.

---

## Enfoque

El estudio se desarrolla de forma progresiva:

1. Ecosistema  
2. Suelo  
3. Microbiología  
4. Árbol  
5. Superficie del árbol  
6. Hoja  
7. Transformación  
8. Transferencia  
9. Sistema acuático

## Idea central

La materia orgánica de origen foliar no es un producto.

Es el resultado de un sistema ecológico y un proceso de transformación.

---

## Relación con SCIENCE

Esta sección conecta:

ecosistema → hoja → compuestos → agua

y se apoya en:

- química de componentes  
- materia orgánica disuelta  
- procesos limnológicos

## Nota
