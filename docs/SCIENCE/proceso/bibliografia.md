# Bibliografía — PROCESO

> **Nota de alcance (importante):**  
> Esta bibliografía sustenta *principios físico-químicos y ecológicos generales* (DOM/NOM, humificación, complejación, estabilidad) aplicables a sistemas naturales y a formulaciones botánicas. **No describe ni valida el proceso interno** de Quercus Flumen de forma replicable (no incluye parámetros operativos exactos ni “recetas”).
