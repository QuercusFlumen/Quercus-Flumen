# TESTS  
## Sistema de validación técnica y científica – Quercus Flumen

---

## 1. Objetivo

Este documento define el sistema de validación técnica aplicado a cada lote de Quercus Flumen.

La finalidad es:

- Garantizar estabilidad físico-química.
- Validar reproducibilidad entre lotes.
- Detectar desviaciones estructurales.
- Aportar trazabilidad medible.
- Permitir futura certificación externa.

No se basa en percepción.
Se basa en parámetros medibles.

---

# 2. Marco de validación

La validación se estructura en 4 niveles:

1. Parámetros físico-químicos básicos.
2. Estabilidad temporal.
3. Integridad organoléptica.
4. Comparativa interlote.

---

# 3. Parámetros físico-químicos básicos

## 3.1 Conductividad eléctrica (µS/cm)

Instrumento:
Medidor EC calibrado.

Objetivo:
Determinar carga iónica total.

Rango esperado Wild V1:
200 – 300 µS/cm

Interpretación:
- Valores elevados pueden indicar arrastre mineral.
- Valores muy bajos pueden indicar baja extracción orgánica.

---

## 3.2 TDS (ppm)

Instrumento:
Medidor combinado EC/TDS.

Rango esperado:
100 – 150 ppm

Nota:
El TDS no distingue composición.
Se usa como indicador comparativo interno.

---

## 3.3 pH

Instrumento:
pH-metro calibrado.

Rango esperado:
5.3 – 5.8

Observación:
La tendencia debe ser estable tras 24h.
No deben observarse oscilaciones bruscas.

---

## 3.4 Temperatura de medición

Registro obligatorio.

Referencia estándar:
10 – 12 °C durante extracción.

---

# 4. Ensayos de estabilidad temporal

Se realizan mediciones en:

- 18h
- 36h
- 48h
- 72h

Parámetros evaluados:

- Conductividad
- TDS
- pH
- Observación visual
- Observación olfativa

---

## 4.1 Criterios de estabilidad aceptables

Un lote se considera estable si:

- Variación EC < 5%
- Variación pH < 0.2 unidades
- No aparece decantación visible
- No aparece turbidez progresiva
- No aparece olor fermentativo

---

# 5. Evaluación visual

## 5.1 Colorimetría cualitativa

- Ámbar translúcido
- No opaco
- No grisáceo
- No negruzco

Ensayo simple:
Columna de 8–11 cm de diámetro → fondo no visible.
Contraluz → transmisión parcial.

---

## 5.2 Decantación

Observación en reposo 72h.

Criterio:
No debe observarse depósito estructural significativo.

Micro-partículas suspendidas son aceptables antes de filtrado.

---

# 6. Evaluación organoléptica

## 6.1 Olor

Perfil esperado:

- Neutro
- Sutil aroma vegetal
- Hoja mojada / bosque frío
- Ausencia total de fermentación

Cualquier nota agria, alcohólica o ácida intensa invalida el lote.

---

# 7. Ensayos comparativos internos

Cada lote se compara con:

- Lote anterior.
- Media histórica.
- Rango de referencia Wild.

Se evalúa:

- Desviación estándar EC.
- Desviación estándar pH.
- Intensidad cromática comparativa.

---

# 8. Proyección de ensayos avanzados (fase futura)

Para futura certificación:

- TOC (Carbono Orgánico Total).
- Análisis espectrofotométrico UV-Vis.
- Fraccionamiento húmico/fúlvico.
- Perfil mineral ICP-OES.
- Recuento microbiológico post pasteurización.

---

# 9. Validación final de lote

Un lote se considera comercializable si cumple:

- Parámetros físico-químicos dentro de rango.
- Estabilidad mínima 72h.
- Ausencia de decantación estructural.
- Perfil olfativo neutro.
- Registro completo de trazabilidad.

---

# 10. Filosofía de validación

No se busca estandarización industrial agresiva.

Se busca:

- Reproducibilidad estructural.
- Estabilidad natural.
- Coherencia entre proceso y resultado.

La validación es parte del producto.
