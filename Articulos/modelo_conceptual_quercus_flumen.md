# Modelo conceptual del sistema Quercus Flumen

Este diagrama resume el modelo conceptual desarrollado a lo largo de la serie:

**Materia orgánica disuelta, agua forestal y estabilidad en sistemas acuáticos**

El modelo representa la interacción entre los principales componentes que estructuran los sistemas acuáticos influenciados por ecosistemas forestales.

Interpretación del modelo
El sistema puede entenderse como una red de procesos interconectados.
1. Fuente primaria
El ecosistema forestal actúa como fuente de materia vegetal que entra en contacto con el agua.
2. Generación de DOM
La lixiviación del material vegetal libera compuestos orgánicos que pasan a formar parte de la materia orgánica disuelta.
3. Transformaciones del sistema
Una vez en el agua, la DOM puede experimentar múltiples transformaciones:
degradación microbiana
procesos fotoquímicos
interacciones químicas
4. Regulación química
Las moléculas orgánicas influyen en:
complejación de metales
acidez del sistema
propiedades ópticas del agua
5. Equilibrio dinámico
La interacción entre procesos biológicos y químicos genera un equilibrio dinámico que permite la estabilidad del sistema.
Naturaleza del modelo
Este modelo no pretende describir un sistema cerrado ni completamente determinista.
Más bien representa una estructura conceptual que ayuda a comprender la interacción entre:
ecosistemas forestales
materia orgánica disuelta
química del agua
microbiología acuática
Contexto dentro de la serie
El modelo conceptual sintetiza los conceptos desarrollados en los siguientes tomos:
El sistema natural
Química del agua forestal
Dinámica del sistema
Modelos y aplicación conceptual
Cada uno de estos bloques aporta elementos que contribuyen a comprender el funcionamiento global del sistema.
