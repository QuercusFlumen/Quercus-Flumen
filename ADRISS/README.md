# Proyecto Quercus Flumen – Dossier ADRISS (LEADER 2023–2027)

Este directorio contiene la estructura completa del dossier técnico,
económico y estratégico preparado para su presentación ante ADRISS
(Asociación para el Desarrollo Rural Integral de las Sierras de Salamanca),
en el marco del programa LEADER 2023–2027.

El proyecto Quercus Flumen propone el desarrollo productivo y validación
científica de un extracto orgánico de hoja de Quercus spp. obtenido en la
Sierra de Francia mediante un proceso controlado y sostenible.

El dossier está estructurado en módulos independientes para facilitar:

• Presentación formal
• Evaluación técnica
• Evaluación económica
• Evaluación ambiental
• Justificación territorial
• Justificación estratégica
