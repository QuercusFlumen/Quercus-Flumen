---

## 📌 OBJETIVO DE ESTA CARPETA

Esta sección documenta de forma estructurada el proyecto **Quercus Flumen** para su evaluación dentro del programa LEADER gestionado por ADRISS.

El proyecto plantea la implantación progresiva de una microindustria rural técnica basada en el aprovechamiento sostenible de hoja caída de Quercus spp., mediante un proceso controlado, documentado y ambientalmente compatible.

# 🧭 ENFOQUE DEL PROYECTO

Quercus Flumen no plantea industrialización intensiva.

Plantea:

- Transformación ligera de biomasa caída
- Producción técnica documentada
- Escalado condicionado
- Compatibilidad ambiental prioritaria
- Microindustria rural especializada

El crecimiento estará siempre supeditado a:
- Validación técnica
- Compatibilidad ambiental
- Demanda real
