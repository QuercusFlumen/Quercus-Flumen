# Quercus Flumen

## Origen forestal. Ciencia real. Resultado tangible.

---

## Un producto que nace del bosque

Quercus Flumen no es una marca creada desde una idea.

Es el resultado de trabajar directamente con:

- hoja forestal
- agua
- sistemas acuáticos reales

Durante años.

No partimos de una fórmula.

Partimos de la observación.

## Wild Core

### Concentrado Orgánico Forestal

Un formato seco que permite generar en casa
un concentrado líquido de alta carga orgánica.

No es una infusión.

Es un sistema.
