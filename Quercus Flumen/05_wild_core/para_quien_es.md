# Para quién es Wild Core

## Un producto que no es para todos

Wild Core no está diseñado para encajar en todos los usos.

No es un producto universal.

Y eso es intencional.

---

## Para quién sí es

### 1. Acuaristas que trabajan con sistemas naturales

Wild Core tiene especial sentido en:

- acuarios de tipo blackwater
- biotopos naturales
- sistemas con carga orgánica controlada

Donde la materia orgánica forma parte del sistema,
no es un elemento externo.

---

### 2. Usuarios que entienden su acuario como un sistema

Personas que:

- observan el comportamiento del agua
- entienden que cada acuario es distinto
- ajustan en función de la respuesta del sistema

Wild Core no da respuestas cerradas.

Funciona mejor cuando el usuario interpreta.

---

### 3. Procesos de maduración

Especialmente útil en:

- acuarios nuevos
- reinicios de sistema
- entornos con baja actividad biológica

Donde se busca activar la base microbiológica.

---

### 4. Cría y recuperación

En sistemas preparados:

- puede favorecer entornos estables
- ayudar a reducir estrés
- mejorar condiciones generales del medio

Siempre desde el entorno,
no desde la intervención directa.

---

### 5. Usuarios que buscan control

Wild Core permite:

- ajustar concentración
- modular intensidad
- adaptar el uso

Es ideal para quien quiere decidir
y no solo aplicar.

---

## Para quién no es

### 1. Usuarios que buscan una dosis exacta universal

Wild Core no funciona así.

No existe una única dosis válida.

Si se busca una solución cerrada,
no es el producto adecuado.

---

### 2. Acuarios altamente artificiales

En sistemas donde:

- la química está completamente controlada
- no se desea introducir variabilidad orgánica

Wild Core puede no tener sentido.

---

### 3. Uso inmediato sin adaptación

No está diseñado para:

- generar efectos instantáneos
- sustituir acondicionadores químicos
- resolver problemas puntuales

---

### 4. Usuarios que no desean experimentar

Wild Core requiere:

- observación
- ajuste progresivo
- comprensión del sistema

Sin esto, su potencial no se aprovecha.

---

## Compatibilidad

Wild Core puede integrarse en muchos sistemas,
pero siempre debe hacerse de forma progresiva.

Es compatible con:

- acuarios plantados (con control)
- sistemas con madera y elementos naturales
- acuarios de cría

Pero requiere adaptación.

---

## Idea clave

Wild Core no simplifica el acuario.

Lo hace más real.

Y eso implica asumir su complejidad.
