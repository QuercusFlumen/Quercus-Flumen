# Wild Core Sachet

## El agua no se tiñe. Se construye.

Wild Core Sachet es la forma funcional y aplicada de Wild Core.

Si Wild Core representa la base, la materia prima y la lógica del sistema, Wild Core Sachet representa su formato de uso más directo, limpio y estructurado.

No es una simple bolsita con materia vegetal.  
No es una infusión.  
No es un aditivo instantáneo.

Es un sistema natural de activación y liberación progresiva diseñado para construir un agua orgánica más cercana a la dinámica de los entornos blackwater naturales.

---

## Qué es Wild Core Sachet

Wild Core Sachet es una unidad de materia orgánica contenida, dosificada y preparada para entrar en el sistema de forma controlada.

Su función no es únicamente aportar color ni liberar compuestos de manera puntual.

Trabaja en dos niveles:

- generar un primer líquido base durante la fase de activación  
- continuar una liberación estable y progresiva una vez introducido en el filtro  

Esto convierte al sachet en algo más que un formato cómodo.

Lo convierte en un sistema orgánico activo.

---

## El objetivo: biotopo real, no efecto visual

Wild Core Sachet no busca producir “agua negra”.

Busca aproximarse a un biotopo blackwater funcional.

Esto implica trabajar con:

- carga orgánica disuelta (DOM) real  
- equilibrio entre fracciones orgánicas  
- dinámica química no lineal  
- integración biológica progresiva  

Un agua oscura no define un blackwater.

Un blackwater real se define por su funcionamiento.

---

## De dónde nace

Wild Core Sachet nace del mismo lugar que Wild Core:

- un entorno dominado por robles  
- observación directa del bosque  
- experimentación prolongada  
- búsqueda de un agua funcional, no solo estética  

La conclusión fue clara:

una hoja no reproduce un sistema  
un extracto no reproduce un proceso  

Wild Core construye la base.  
Wild Core Sachet permite aplicarla.

---

## 100% natural y ecológico

Wild Core Sachet parte exclusivamente de materia orgánica natural.

No depende de:

- colorantes  
- extractos industriales agresivos  
- fracciones químicamente forzadas  
- procesos de síntesis  

Su enfoque es ecológico en sentido real:

- selección de hoja  
- integración de distintos estados de maduración  
- activación física y térmica  
- evolución posterior dentro del sistema  

No se trata de una materia plana.

Se trata de una materia que conserva su lógica de origen.

---

## La lógica de fondo: una hoja no basta

Una sola fase de hoja no representa el sistema completo.

Cada estado aporta algo distinto:

- hojas menos maduras → mayor reactividad  
- fases intermedias → mayor movilidad  
- fases más transformadas → mayor estructura  

Wild Core Sachet integra estas fases para construir un perfil más completo, equilibrado y coherente.

---

## Un perfil orgánico más completo

Wild Core Sachet no busca maximizar una sola dimensión.

No está diseñado para:

- generar el máximo color  
- concentrar una única fracción  
- producir un impacto inmediato  

Busca un equilibrio entre:

- fracción tánico-fenólica (reactividad)  
- fracción fúlvica-like (movilidad)  
- fracción húmica-like (estructura no dominante)  

Ese equilibrio es lo que define su comportamiento.

---

## No solo química: también biología

Wild Core Sachet no es solo una fuente de compuestos.

Su comportamiento incluye tres niveles:

### 1. Nivel químico
Liberación inicial de materia orgánica soluble y activa.

### 2. Nivel fisicoquímico
Evolución del sistema en el tiempo (color, pH, conductividad).

### 3. Nivel biológico
Colonización bacteriana y degradación aeróbica dentro del filtro.

Esto significa que el sachet no introduce solo materia.

Introduce un proceso.

---

## Formato limpio, pero funcional

El formato sachet permite:

- dosificación clara  
- uso limpio  
- integración sencilla en el filtro  

Pero su valor no está en la comodidad.

El sachet actúa como una interfaz.

Pasa de:

- estado compacto (seco)  

a:

- estado expandido, permeable y activo (tras activación)  

Ese cambio físico es parte del sistema.

---

## La activación es obligatoria

Wild Core Sachet no se utiliza en seco.

Debe activarse previamente.

Durante la activación:

- la materia se hidrata completamente  
- se elimina el aire interno  
- comienza la liberación de compuestos solubles  
- se genera un líquido base funcional  

Este líquido no es un subproducto.

Es parte del sistema.

Puede utilizarse directamente para trabajar sobre agua nueva.

---

## Sistema de doble fase

Wild Core Sachet trabaja en dos tiempos:

### Fase 1 — Activación
Generación de un líquido base orgánico activo.

### Fase 2 — Integración en filtro
Liberación progresiva y estabilización del sistema.

Después de la fase inicial, el sachet no se agota de forma brusca.

Cambia de función.

---

## Doble vida funcional

Wild Core Sachet presenta dos etapas:

### Fase química
Impacto inicial mediante liberación de compuestos solubles.

### Fase biológica
Integración como materia orgánica estructural y soporte bacteriano.

No deja de funcionar.

Evoluciona.

---

## Referencia y variabilidad

El sistema se ha trabajado con referencia en acuarios tipo de 60 litros.

Sin embargo, su comportamiento depende de:

- tipo de filtro  
- caudal  
- oxigenación  
- temperatura  
- carga biológica  

Por eso:

no se definen valores fijos  
se trabaja con rangos y comportamiento real  

---

## Seguridad y estabilidad

En condiciones adecuadas:

- buen flujo  
- buena oxigenación  
- ausencia de compactación  

el sachet no se pudre.

Se degrada de forma aeróbica y estable.

El filtro no es neutro.

Define el comportamiento del sistema.

---

## Qué hace diferente a Wild Core Sachet

No es diferente por el formato.

Es diferente porque integra:

- materia orgánica compleja  
- activación controlada  
- liberación progresiva  
- integración biológica  
- continuidad en el sistema  

No ofrece un efecto.

Ofrece un proceso.

---

## Idea clave

Wild Core Sachet no simplifica el bosque.

Organiza parte de su complejidad.

---

## Conclusión

Wild Core Sachet es una herramienta para construir agua.

No actúa solo al inicio.  
No actúa solo en lo visible.  
No actúa solo a nivel químico.

Activa, integra y evoluciona.

Porque el agua no se tiñe.

Se construye.
