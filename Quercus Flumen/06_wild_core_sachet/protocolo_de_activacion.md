# Protocolo de activación — Wild Core Sachet

## Introducción

Wild Core Sachet no está diseñado para una extracción completa previa.

Está diseñado para ser activado.

La activación es el proceso que permite que la materia orgánica pase de un estado seco a un estado funcional dentro del sistema.

---

## Objetivo

El objetivo del proceso no es agotar el contenido del sachet.

Es:

- hidratar completamente el material  
- eliminar el aire interno  
- iniciar la liberación de compuestos solubles  
- preparar la bolsa para su uso en el filtro  

---

## Procedimiento

1. Introducir el sachet (5 g) en aproximadamente 0,5 L de agua  

2. Iniciar calentamiento desde temperatura ambiente  

3. Llevar a una cocción suave (burbujeo ligero)  

4. Mantener durante aproximadamente 5 minutos  

---

## Qué ocurre durante la activación

Durante este proceso:

- el material se hidrata completamente  
- el aire interno es desplazado  
- comienza la liberación de fracciones solubles  
- se genera un primer líquido base  
- la materia orgánica entra en estado activo  

---

## Resultado

Tras este proceso, el sachet:

- está completamente hidratado  
- ha iniciado su actividad química  
- está listo para ser introducido en el filtro  

El líquido generado puede ser utilizado directamente.

---

## Importante

Este proceso no busca una extracción completa.

Una extracción excesiva:

- reduce la capacidad del sachet en el filtro  
- altera el equilibrio del sistema  

---

## Idea clave

El objetivo no es extraer todo.

Es activar la materia para que funcione dentro del sistema.

---

## Conclusión

Wild Core Sachet no se utiliza en estado seco.

Se activa.

Y esa activación es la que permite que el sistema funcione de forma estable y coherente a lo largo del tiempo.
