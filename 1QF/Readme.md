# Quercus Flumen  
## El agua no se tiñe. Se construye.

Quercus Flumen es un proyecto centrado en la comprensión y aplicación
de la materia orgánica en sistemas blackwater.

No nace como un producto.

Nace de un entorno, de un proceso y de una forma de entender el agua.

---

## Qué vas a encontrar aquí

Este repositorio recoge tres pilares:

- la base del proyecto  
- el sistema Wild  
- la base científica que lo respalda
