# Testaje  
## Especies de hoja

Durante el desarrollo del sistema Quercus Flumen se realizaron pruebas comparativas utilizando diferentes especies de hoja presentes en el mismo entorno natural.

El objetivo fue evaluar cómo influye la especie vegetal en:

- capacidad de extracción
- intensidad de color
- estabilidad del extracto
- generación de residuo orgánico

Las pruebas se realizaron utilizando hojas procedentes del mismo entorno forestal para reducir al máximo la influencia de factores externos.

Las especies evaluadas fueron:

- Roble (*Quercus*)
- Castaño (*Castanea*)
- Plátano de sombra (*Platanus*)

---

# Hoja de roble

La hoja de roble mostró el comportamiento más equilibrado en las pruebas realizadas.

Las observaciones principales fueron:

- buena capacidad de extracción
- color ámbar estable
- baja generación de residuo orgánico
- comportamiento consistente tras periodos prolongados de reposo

El extracto obtenido presentaba un equilibrio notable entre intensidad de color, estabilidad del líquido y limpieza del sistema.

Este comportamiento se repitió de forma consistente en diferentes estados de maduración de la hoja.

---

# Hoja de plátano de sombra

La hoja de plátano de sombra mostró una **capacidad de tintado inicial más alta**.

Las observaciones principales fueron:

- liberación de color más rápida
- tonalidades más oscuras en menor tiempo
- extracción inicial intensa

Sin embargo, el extracto obtenido mostró una menor estabilidad a largo plazo y una fracción orgánica menos equilibrada en comparación con las extracciones realizadas con hoja de roble.

El sistema tendía a producir un comportamiento más variable tras periodos prolongados de reposo.

---

# Hoja de castaño

La hoja de castaño presentó un comportamiento diferente al de las otras especies evaluadas.

Las observaciones más destacadas fueron:

- degradación más rápida de la hoja
- menor intensidad de color
- mayor generación de materia orgánica particulada

Aunque la hoja se degradaba con facilidad durante el proceso, la fracción orgánica soluble obtenida resultó menos intensa y menos estable que en las pruebas realizadas con hoja de roble.

Este comportamiento generaba un extracto con mayor presencia de residuos y menor estabilidad general.

---

# Conclusión del testaje

Las pruebas realizadas mostraron que **no todas las especies de hoja generan el mismo tipo de extracto**.

Algunas especies pueden producir color intenso de forma rápida, mientras que otras se degradan con facilidad generando mayor cantidad de residuo orgánico.

Entre las especies evaluadas, la hoja de roble mostró el comportamiento más equilibrado entre:

- capacidad de extracción
- estabilidad del extracto
- limpieza del sistema
- consistencia a largo plazo

Por esta razón se continuó el desarrollo del sistema Quercus Flumen utilizando hoja de roble como materia prima principal.
