ANALÍTICA PARA VALIDAR LA CALIDAD DEL EXTRACTO WILD
Documento técnico resumido

1. OBJETIVO

Confirmar científicamente:

- seguridad química
- carga orgánica real (DOM)
- perfil óptico de la materia orgánica
- estabilidad microbiológica
- comportamiento en sistema acuático


--------------------------------------------------
2. PAQUETE ANALÍTICO MÍNIMO (LABORATORIO)
--------------------------------------------------

A. Carbono orgánico disuelto (DOC)

Permite medir cuánta materia orgánica real aporta el extracto.


B. Espectros UV-Vis (200-700 nm)

Parámetros clave:

- Abs254
- SUVA254 = Abs254 / DOC
- Abs420-440 (color aparente)
- E2/E3 = Abs250 / Abs365 (proxy peso molecular)

Interpretación general:

SUVA254 bajo → DOM menos humificado, más fúlvico y reactivo  
SUVA254 alto → DOM más aromático/húmico


C. Parámetros básicos de agua

- pH
- Conductividad
- Alcalinidad
- GH / KH


D. Metales traza (ICP-OES o ICP-MS)

Especial atención a:

- Cu
- Zn
- Fe
- Mn
- Al

Esto confirma seguridad en aguas blandas.


E. Microbiología básica

- Recuento bacterias heterótrofas (HPC)
- Levaduras / mohos si el laboratorio lo ofrece

Sirve para evaluar estabilidad del producto.


--------------------------------------------------
3. COMPARACIÓN EXPERIMENTAL RECOMENDADA
--------------------------------------------------

Para obtener resultados interpretables se recomienda analizar:

1. Wild
2. Agua control (RO o agua base)
3. Extracto comercial negro
4. Infusión estándar de hojas (preparación control)

IMPORTANTE:

Las comparaciones deben hacerse a igual DOC,
no a igual dosis en ml.


--------------------------------------------------
4. RESULTADOS ESPERADOS SI WILD TIENE PERFIL DOM LIGERO
--------------------------------------------------

Si Wild está dominado por fracciones fúlvicas y polifenoles ligeros:

- SUVA254 moderado o bajo
- Relación E2/E3 relativamente alta
- Menor color por unidad de DOC frente a extractos negros
- Metales muy bajos
- Baja turbidez y buena estabilidad

Esto indicaría DOM joven y reactivo,
más cercano a lixiviados forestales naturales.


--------------------------------------------------
5. PARÁMETROS PARA ACUARIOFILIA
--------------------------------------------------

Evaluar además tras dosificación:

- NH4 / NH3
- NO2
- NO3
- PO4

El extracto no debería introducir nutrientes significativos.


--------------------------------------------------
6. ENSAYO BIOLÓGICO SIMPLE
--------------------------------------------------

Montar 3 acuarios idénticos:

1. Control
2. Wild
3. Extracto comercial oscuro

Monitorear durante 4-6 semanas:

- pH
- Conductividad
- NH3/NH4
- NO2
- NO3
- estabilidad del color
- comportamiento peces/gambas
- desarrollo de biopelícula


--------------------------------------------------
7. CONCLUSIÓN
--------------------------------------------------

Si Wild demuestra:

- DOC medible
- perfil óptico coherente
- baja presencia de metales
- estabilidad microbiológica
- comportamiento estable en acuario

entonces el producto queda científicamente validado
como extracto orgánico funcional para sistemas
de tipo blackwater.
