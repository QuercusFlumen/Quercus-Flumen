# Productos analizados

Esta carpeta contiene las fichas individuales de productos comerciales analizados dentro del estudio de competencia del proyecto **Wild / Quercus Flumen**.

Cada archivo describe un producto concreto del mercado de:

- extractos blackwater
- acondicionadores amazónicos
- extractos húmicos líquidos
- extractos botánicos líquidos

El objetivo de estas fichas es analizar de forma sistemática:

- qué información analítica proporcionan
- qué efectos declaran
- qué tipo de dosificación recomiendan
- qué composición química es plausible
- qué nivel de transparencia existe en el sector

Estas fichas forman parte de la investigación interna del proyecto.

El objetivo **no es desprestigiar a ningún fabricante**, sino entender con precisión el funcionamiento real del mercado.

# Orden de los archivos

Los productos se numeran para mantener un orden estable dentro del repositorio.

El número **no indica calidad ni clasificación**, simplemente orden.
